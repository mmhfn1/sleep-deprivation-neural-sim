§ 01 — Abstract

What is this and why does it exist?

Sleep deprivation is one of the most studied — and most ignored — threats to human cognitive performance. This simulation makes the invisible visible.

The Problem

After 24 hours without sleep, most people feel impaired but dramatically underestimate how impaired they actually are. After 48 hours, the brain starts hallucinating. After 72, the risk of severe cognitive failure, microsleeps behind the wheel, and emotional dysregulation reaches critical levels. Neuroscience has modelled this for decades — but the models live in academic papers, not interactive tools anyone can actually use.

The Approach

This simulation implements Borbély's 1982 two-process model — the same mathematical framework used in real sleep medicine research — and translates it into per-neuron firing rates across five brain regions using biologically-grounded parameters. A 60 FPS animated neural canvas makes the abstract data visceral. A Phase 2 pharmacology layer models how caffeine and nicotine interact with the sleep-deprived brain, including the caffeine crash, nicotine tolerance, and stimulant washout after 48 hours.

The anti-flatline guarantee

A key design requirement was that the simulation must never "flatline" — no region's firing rate can reach zero. This is achieved through a three-layer defence: the ODE for sleep pressure dS/dt = (1−S)/τ_w is asymptotic (S approaches 1.0 but never reaches it), per-region rates use a tanh soft-floor formula rather than a linear clamp, and every region has a hard biological floor at 8–12% of baseline — because in reality, neurons don't go silent until the brain shuts down entirely.

§ 02 — The Science

Two processes, one brain

The Borbély two-process model (1982) describes sleep regulation as the interaction of two independent oscillators. It's been validated across four decades of sleep research.

Process S — Homeostatic Pressure

# Adenosine accumulates while awake dS/dt = (1 − S) / τ_w τ_w = 18.2 h S(0) = 0.20 # morning baseline # Analytical solution (used here) S(t) = 1 − 0.80 · exp(−t / 18.2) S(72h) ≈ 0.982 # never reaches 1.0

Process C — Circadian Signal

C(t) = 0.5 · [1 + cos(2π(t − φ) / 24)] φ = 14 h # peak wake-drive at ~10 pm # (the SCN "evening forbidden zone")

Combined Sleep Propensity W

W(t) = clip(0.85·S − 0.40·C, 0, 1) W → NORMAL when W < 0.25 W → MILD when W < 0.45 W → MODERATE when W < 0.60 W → SEVERE when W < 0.75 W → CRITICAL when W ≥ 0.75
Why the ODE approach?

Using a differential equation (rather than a lookup table) means S is a continuous, smooth function with mathematically provable bounds. It literally cannot reach 1.0 — the exponential decay term prevents it.

The circadian paradox

Despite massive sleep pressure at 24h, most people feel a second wind in the evening. That's C doing its job — the SCN fights sleep timing to enforce a consistent schedule. This is modelled here as an oscillating 24h signal.

Firing rate formula

Each brain region converts W into a firing rate through a tanh soft-suppression:

rate = f_floor + (f_base − f_floor) × (1 − 0.92 · tanh(|α| · W)) # f_floor = 8–12% of baseline # α controls sensitivity (see regions)
Amygdala exception

The amygdala is modelled with α = −0.50 (negative), causing it to increase under sleep pressure. This captures the well-documented emotional dysregulation, hypervigilance, and anxiety that accompany sleep deprivation.

§ 03 — Brain Regions

Five regions, five stories

Each region has its own baseline firing rate, sensitivity to sleep pressure (α), and biological response. No two behave the same way under deprivation.

Prefrontal Cortex

35 Hz baseline
α = +0.90 · ε = 8% floor

Executive function, decision-making, impulse control. Most sensitive to sleep loss. First to go, last to come back.

↓ Steepest decline

Hippocampus

28 Hz baseline
α = +0.75 · ε = 8% floor

Memory encoding and consolidation. Rapid degradation around 24h explains why sleep-deprived people struggle to form new memories.

↓ Memory collapse ~24 h

Thalamus

40 Hz baseline
α = +0.65 · ε = 10% floor

The sensory relay station and sleep gating hub. After 28h, microsleep intrusions (4–12 frame bursts of 95% suppression) are injected stochastically.

↓ Microsleep onset 28 h

Amygdala

22 Hz baseline
α = −0.50 · (inverse response)

Emotional processing and threat detection. Uniquely increases under sleep deprivation. Explains heightened irritability, anxiety, and impulsive emotional reactions.

↑ Hyperactivates

Brainstem RAS

30 Hz baseline
α = +0.45 · ε = 12% floor

The reticular activating system — the brain's arousal engine. Most resilient; compensates for cortical decline until ~48h, then concedes.

→ Compensates to 48 h

What happens, hour by hour

0 h

Normal function

All regions firing at baseline. W ≈ 0.05. Full executive function, intact memory encoding, stable emotional responses.

16 h

Mild degradation begins

PFC starts declining measurably. Hippocampal encoding slows. Performance similar to 0.05% blood alcohol. Most people feel "a little tired" but significantly underestimate their impairment.

28 h

Microsleeps begin

The thalamus starts involuntary 80–200ms blackout events. You're awake but functionally asleep for fractions of a second. Driving is now genuinely dangerous. Amygdala hyperactivation is clearly measurable.

48 h

Severe territory

Brainstem RAS begins conceding. PFC at ~50% of baseline. Hallucinations may begin. Emotional regulation severely compromised. Stimulant efficacy drops to 30% of normal levels as adenosine saturation overwhelms receptor blockade.

72 h

Critical

The model reaches its full extent. S ≈ 0.982 — asymptotic maximum. W > 0.80. All cortical regions suppressed to 40–55% of baseline. Microsleep probability peaks at ~4% per frame. Amygdala is 65% above baseline.

§ 04 — Phase 2: Pharmacology

What happens when you add caffeine?

The pharmacology layer models how real substances interact with the sleep-deprived brain using industry-standard PK/PD modelling — the same approach pharmaceutical companies use for drug design.

  • Adenosine receptor antagonist — blocks the mechanism that accumulates sleep pressure
  • S_modifier = −(E_max · C) / (EC₅₀ + C), reducing effective sleep pressure by up to 60% at peak plasma concentration
  • Crash: when plasma drops below 10% of peak, adenosine rebounds with +0.12 added to S — the "caffeine crash"
  • High dose (>400 mg cumulative): amygdala anxiety effect +15% — jitteriness, restlessness
  • After 48 h: stimulant boost decays to ≤30% efficacy as adenosine saturation overwhelms receptor blockade

  • Tolerance: E_eff = E_max × exp(−0.15 × n_prev_doses) — each subsequent dose is less effective
  • Synergy with caffeine: extra 5% amygdala anxiety when both are active simultaneously
  • Very short half-life means effects dissipate quickly between doses — interval dosing needed

How to use interval dosing

In the substance panel, you can set a dose schedule instead of a single dose. For example: 200mg caffeine every 4h from hour 8 to hour 48 generates 11 individual doses at hours 8, 12, 16, 20, 24, 28, 32, 36, 40, 44, and 48. The simulation recomputes all 73 hour snapshots with the full cumulative pharmacokinetics. You can then scrub to any hour and see exactly how the firing rates differ from the no-drug baseline.

§ 05 — Using the Simulation

Every button, explained

The interface is split into four zones: the neural canvas (left), charts and gauges (right), playback controls (bottom bar), and the pharmacology panel (very bottom).

Neural Canvas

200 animated neurons (40 per region) firing in real time at 60 FPS. Node brightness reflects the Poisson spike probability at the current hour's firing rate. Radial pulses expand from each spike. The red vignette intensifies with sleep pressure W. A dark indigo tint over the thalamus marks a microsleep event.

Play >> / || Pause

Plays the simulation from the current hour forward. Each hour advances at the rate set by the Speed buttons. At 1x = 1 hour per second (72s for full run). At 10x = 7 seconds. Playback auto-pauses at hour 72.

|< and >| (Step buttons)

Step backward or forward by the amount selected in the Step size buttons: 1h, 6h, 12h, or 24h. Use 6h steps for a quick overview; 1h steps when examining specific transitions around hour 16, 28, or 48.

Step: 1h / 6h / 12h / 24h

Controls how far each |< and >| click jumps. Select 24h to quickly jump across major milestones. Select 1h when examining fine-grained pharmacology effects around the caffeine peak (~3h after dose).

Speed: 1x / 2x / 5x / 10x

Hours per second during playback. Each speed button shows a tooltip explaining the rate. 5x is a good balance for watching circadian waves emerge. 10x for a rapid full-run overview.

Hour slider

Scrub to any hour from 0 to 72. All widgets update instantly — the neural canvas animation changes to reflect the new hour's firing rates, the gauge needle moves, and the time-series cursor jumps.

Sleep Pressure Gauge

Shows the W value (combined sleep propensity) as an arc from 0% to 100%. The needle position and the label (NORMAL / MILD / etc.) always agree with the header status — no confusion between different metrics.

Firing Rate Chart

PyQtGraph time-series chart showing all 5 regions across 0–72h. Pre-loaded at startup, so scrubbing is instant. The white dashed cursor line tracks the current hour. Circadian waves are visible as regular oscillations.

Export CSV

Saves a 73-row CSV with columns for hour, S, C, W, impairment_label, impairment_index, caff_conc, nic_conc, and firing rates for all five regions. Useful for further analysis in Python or a spreadsheet.

Caffeine / Nicotine (Phase 2)

Enable either checkbox to reveal interval dosing controls: Amount, Every N hours, From hour, To hour. Example: "200mg caffeine every 4h from 8h to 48h" → 11 doses. Hit Apply Doses to recompute.

Apply Doses

Triggers a full recompute of all 73 snapshots with the pharmacology schedule applied. Takes ~20ms on modern hardware. The chart updates immediately with the new curves. Toggle substances off and Apply again to return to baseline.

Reset

Rewinds to hour 0, clears all pharmacology, and recomputes. The canvas animation resets to baseline firing rates. All region bars return to their resting values. Phase 2 checkboxes are unchecked.

§ 06 — Getting Started

Running it locally

Three commands. No Docker, no virtual environment dance required (though using one is always good practice).

1

Clone the repo

$ git clone https://github.com/mmhfn1/sleep-deprivation-neural-sim.git $ cd sleep-deprivation-neural-sim
2

Install dependencies

Requires Python 3.10+. On Linux you may need libgl1 installed (sudo apt-get install libgl1).

$ pip install -r requirements.txt
3

Run the simulation

$ python run.py

The window opens at 1640×980. Minimum 1400×860. If you're on a small screen, the layout scales down but all functionality works.

4

Run the test suite (optional but recommended)

$ python tests/test_two_process.py # 10 model tests $ python tests/test_spike_generator.py # 12 region + spike tests $ python tests/test_pharmacology.py # 11 PK/PD tests

Dependencies

PyQt6 ≥ 6.6 — GUI framework and 60 FPS canvas  ·  pyqtgraph ≥ 0.13 — hardware-accelerated time-series plot  ·  numpy ≥ 1.26 — vectorised spike generation  ·  scipy ≥ 1.12 — available for signal processing extensions. No matplotlib anywhere.
